# ポートフォリオ

このリポジトリは、Web開発者のポートフォリオサイトです。

## 構成

- `index.html`: メインのHTMLファイル
- `styles.css`: スタイリング用のCSSファイル
- `script.js`: インタラクション用のJavaScriptファイル
- `images/`: 作品のスクリーンショットを保存するフォルダ
